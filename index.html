Reconnaissance

Tools Utama:

  • Nmap - Network scanning
  • theHarvester - OSINT gathering
  • Shodan - Search engine untuk devices
  • Maltego - Data mining dan visualization

Web Security

Testing keamanan aplikasi web dengan berbagai teknik seperti SQL injection, XSS, CSRF, dan lainnya.

Level: Menengah Tools: 15 Modul: 20

Tools Utama:

  • Burp Suite - Web vulnerability scanner
  • OWASP ZAP - Open source scanner
  • SQLmap - Automated SQL injection
  • Nikto - Web server scanner

Level: Pemula - Lanjutan Tools: 18 Modul: 25

Tools Utama:

  • Wireshark - Network protocol analyzer
  • Metasploit - Penetration testing framework
  • Aircrack-ng - Wireless network security

4 Minggu 10 Modul 6 Projects

Burp Suite OWASP ZAP SQLmap Nikto Dirb/Dirbuster

Tools yang Dipelajari:

Metasploit Framework Empire Cobalt Strike PowerShell Empire Mimikatz

Wireless & Mobile Security

Aircrack-ng Wifite MobSF Frida Burp Suite Mobile

2. Akun Pengguna

Ethical Guidelines

Prinsip Ethical Hacking:

  • Authorization Selalu: Selalu dapatkan izin tertulis sebelum menguji sistem apapun
  • Laporan yang Bertanggung Jawab: Laporkan kerentanan kepada pemilik sistem dengan cara yang bertanggung jawab
  • Jangan Membahayakan: Hindari tindakan yang dapat menyebabkan kerusakan atau gangguan layanan
  • Hormati Privasi: Jangan mengakses, mengubah, atau menghapus data tanpa izin
  • Transparansi: Jadilah transparan tentang kemampuan dan keterbatasan Anda

Kode Etik Profesional:

  • Utamakan keamanan dan privasi pengguna
  • Jaga kerahasiaan informasi sensitif
  • Laporkan aktivitas ilegal yang Anda temukan
  • Terus tingkatkan pengetahuan dan keterampilan
  • Berkontribusi pada komunitas cybersecurity
